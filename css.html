CSS ieviešana

HTML sākotnēji nebija paredzēts, lai tas saturētu formatēšanas tagus. Tas tika radīts, lai aprakstītu tīmekļa lapu saturu. CSS tika ieviests 1996. gadā un ļauj kontrolēt, kā HTML elementi izskatās uz ekrāna. Izmantojot CSS, varam mainīt krāsas, fontus un izkārtojumu, padarot tīmekļa lapas pievilcīgākas un vieglāk lietojamas. CSS ir attīstījies no 1. versijas līdz mūsdienām, pievienojot jaunas funkcijas un uzlabojumus.

Tomēr laika gaitā tīmekļa izstrādātājiem radās problēmas ar stilu formatējumu. Kad HTML3.2 specifikācijai tika pievienoti tādi tagi kā <font> un krāsu atribūti, tas kļuva par īstu izaicinājumu. Lielu tīmekļa vietņu izstrāde prasīja daudz laika un resursu, jo katrai lapai bija jānorāda fonti un krāsas atsevišķi.

Lai risinātu šīs problēmas, tika ieviests CSS (Cascading Style Sheets). CSS ļāva noņemt stila formatējumu no HTML lapām, tādējādi izstrādātājiem bija iespēja atdalīt saturu no dizaina. Tas ievērojami atviegloja tīmekļa lapu izskata maiņu, jo tagad ir iespējams mainīt dizainu, neietekmējot katru lapu atsevišķi. CSS ne tikai uzlaboja tīmekļa vietņu izskatu, bet arī padarīja tās vieglāk uzturamas un efektīvākas.

CSS papildina HTML, radot vizuālo noformējumu tīmekļa vietnei, pielāgojot tās izskatu atkarībā no ierīces, kurā vietni atvērs.

Atceries, ka CSS stilus parasti labāk definēt ārējā vai iekšējā stilā.

CSS sintakse: (Deklarācija: īpašība + vērtība, deklarācijas atdala ar semikolu). Izmantosi, kad vairākiem elementiem būs vienāds noformējums.

Termini no angļu valodas: selector - atlasītājs, declaration - deklarācija, property - īpašība, value - vērtība, attributes - atribūti.
